# Shadows
A 2d platformer game based on switching between the player and its shadow to explore new places.
(the game is a bit short but the main logic and ui is done which makes the next levels much easier to make)

I made this game using godot engine and willing to put much effort to improve it until it's released on steam.
I started by coding the basic player movement and animations. Then created the tilemaps and designed the level. After this I created a collectable object called dark core which will be edited and used later to give more abilities. The dark core works by changing a bool in a global script that is responsible for allowinng the player to use abilities, like wall jump, double jump, dash, switch to shadow and more.
## Controls
WASD - arrows to move around, space to jump and q/shift to dash!
That's all the controls until now, there will be more abilities in this game with more controls so stay tuned for the next part.
Note: Some controls may not be mentioned here and the user will explore it in the game after gaining abilities.
