# Shadows
A 2d platformer game based on switching between the player and its shadow to explore new places.
(the game is a bit short but the main logic and ui is done which makes the next levels much easier to make)

## Controls
WASD - arrows to move around, space to jump and q/shift to dash!
That's all the controls until now, there will be more abilities in this game with more controls so stay tuned for the next part.
